About Stock Market Trend Analysis

Our Mission

Our mission is to provide investors with powerful, easy-to-use tools for analyzing stock market trends and making informed investment decisions.

Features
  • Real-time stock data from Yahoo Finance
  • Interactive stock charts and visualizations
  • Technical indicators (Moving Averages, MACD, RSI)
  • Stock price prediction using machine learning
  • User authentication and personalized watchlists
Technology Stack

This application is built using the following technologies:

  • Backend: Flask (Python)
  • Database: SQLAlchemy with SQLite
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Data Visualization: Plotly
  • Data Analysis: Pandas, NumPy, Scikit-learn
