YAMS - Stock Market Trend Analysis

A powerful web application for analyzing stock market trends, visualizing data, and making predictions.

About the Project

YAMS (Yet Another Market Scanner) is a Flask-based web application that provides tools for stock market analysis. It offers real-time data, interactive visualizations, and machine learning predictions to help users make informed investment decisions.

Real-time Data

Access up-to-date stock market data using yfinance integration.

Interactive Charts

Visualize stock performance with interactive charts and technical indicators.

ML Predictions

Leverage machine learning algorithms to predict potential market trends.
